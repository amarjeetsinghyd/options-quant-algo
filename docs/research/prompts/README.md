# Prompt Corpus

This directory serves as the permanent research corpus for the institution's intellectual history.
It stores the exact prompts that directly contributed to architectural breakthroughs.

## Required Format for Each Entry:

*   **Title:** (e.g., Canonical Observation Dataset Inception)
*   **Objective:** (What was the goal of the prompt?)
*   **Date:** (YYYY-MM-DD)
*   **Model Used:** (e.g., GPT-4, Claude-3)
*   **Outcome:** (Brief summary of the architectural decision reached)
*   **Resulting Document:** (Link to the frozen spec or architecture doc)

*Save each breakthrough as a separate `.md` file in this directory.*
